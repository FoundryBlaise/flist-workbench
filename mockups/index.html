Current scope
The app is a BBCode editor + log browser + RAG chat. No F-list auth or API editing. The user copies BBCode from F-list, edits in F-list Workbench, copies it back. The flistcharexporter userscript stays as a separate, optional companion. RAG indexes F-Chat logs and can fetch any public F-list profile on demand.
